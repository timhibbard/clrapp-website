# Christian Radio Finder Website

Marketing website for the Christian Radio Finder iOS app.

## Live Site

The site is automatically deployed to GitHub Pages at: `https://timhibbard.github.io/clrapp-website/`

## Privacy & Terms Pages for App Store

- **Privacy Policy**: `https://timhibbard.github.io/clrapp-website/privacy.html`
- **Terms of Service**: `https://timhibbard.github.io/clrapp-website/terms.html`

Use these URLs when submitting the app to the App Store.

## GitHub Pages Setup

1. Go to your repository Settings → Pages
2. Set Source to "GitHub Actions"
3. The site will automatically deploy on every push to `main`

## Custom Domain (Optional)

To use a custom domain:

1. Create a `CNAME` file with your domain (e.g., `christianradiofinder.com`)
2. Add the file to the repository root
3. Configure DNS settings with your domain provider
4. Enable "Enforce HTTPS" in GitHub Pages settings

## Local Development

Simply open `index.html` in a browser. No build process required.
